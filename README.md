# Shine Jewel Tech - Industrial Machines Website

A modern, responsive React.js website for Shine Jewel Tech showcasing industrial machines and equipment.

## Features

- Single-page application with smooth scrolling
- Responsive design for all devices
- Product showcase with interactive cards
- Contact form with validation
- Modern UI/UX design

## Deployment

This website is ready to be deployed on Vercel:

1. Push code to GitHub
2. Connect your GitHub repo to Vercel
3. Deploy automatically

## Local Development

```bash
npm install
npm run dev
